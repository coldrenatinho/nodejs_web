🧪 Teste da API de Data/Hora

Esta página demonstra como consumir a API do servidor de data/hora.

Clique em "Testar API" para ver os dados...

📝 Exemplo de Código

// JavaScript - Fetch API fetch('/api/datetime') .then(response => response.json()) .then(data => { console.log('✅ Sucesso:', data); console.log('📅 Data:', data.data.data); console.log('🕐 Hora:', data.data.hora); }) .catch(error => { console.error('❌ Erro:', error); }); // jQuery (se disponível) $.get('/api/datetime') .done(function(data) { console.log('✅ Data recebida:', data); }) .fail(function() { console.log('❌ Erro na requisição'); });
